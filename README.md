# Vrinda Store Annual Sales Analysis Dashboard (Excel)

## Overview

This project analyzes the annual sales performance of **Vrinda Store** using Microsoft Excel. The dashboard provides interactive insights into sales trends, customer demographics, order status, geographic performance, and sales channels.

The objective of this project is to transform raw sales data into a dynamic and visually appealing business dashboard that supports data-driven decision-making.

## Project Objectives

- Analyze annual sales performance.
- Identify top-performing states and sales channels.
- Understand customer demographics by gender and age group.
- Track monthly sales and order volume trends.
- Monitor order fulfillment and return status.
- Provide interactive filtering for detailed analysis.

## Key Business Questions Answered

1. Which months generated the highest sales?
2. What is the relationship between order volume and revenue?
3. Which gender contributes more to sales?
4. Which age group places the most orders?
5. Which states generate the highest revenue?
6. Which sales channels drive the most orders?

## Dashboard Features

### Monthly Orders vs Sales
- Compares order count and revenue across all months.
- Helps identify seasonal sales patterns.

### Sales by Gender
- Visual breakdown of sales contribution by men and women.
- Women contribute approximately **64%** of total sales.

### Order Status Analysis
- Tracks:
  - Delivered Orders
  - Returned Orders
  - Cancelled Orders
  - Refunded Orders

### Top 5 States by Sales
- Highlights highest revenue-generating states.
- Supports regional performance analysis.

### Orders by Age and Gender
- Segments customers into:
  - Adult
  - Teenager
  - Senior
- Compares purchasing behavior between male and female customers.

### Orders by Sales Channel
- Analyzes channel performance across:
  - Amazon
  - Myntra
  - Flipkart
  - Ajio
  - Meesho
  - Nalli
  - Others

### Interactive Slicers
Users can dynamically filter the dashboard by:

- Category
- Sales Channel
- Month

## Key Insights

### Customer Demographics
- Women account for the majority of purchases.
- Adult customers represent the largest customer segment.

### Order Fulfillment
- Approximately **92%** of orders are successfully delivered.
- Return, cancellation, and refund rates remain low.

### Sales Channels
- Amazon is the leading sales channel.
- Myntra and Flipkart are significant contributors to total orders.

### Regional Performance
Top-performing states include:
1. Maharashtra
2. Karnataka
3. Uttar Pradesh
4. Telangana
5. Tamil Nadu
